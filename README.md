# 台女指數測驗 💅

一個互動式網頁測驗，根據 2025/11 DCard 與 Threads 拜金標準來測試「台女指數」。

## 功能特色

- 20 道精心設計的測驗題目
- 華麗的視覺效果與動畫
- 即時進度追蹤
- 個性化的結果分析
- 響應式設計，支援行動裝置
- 鍵盤快捷鍵支援 (1/Y 和 2/N)

## 線上體驗

直接開啟 `index.html` 即可使用，無需伺服器環境。

## 如何新增更多問題

### 步驟 1: 編輯 script.js

找到 `script.js` 檔案中的 `questions` 陣列（第 2-23 行）：

```javascript
const questions = [
    "你會不會沒有買一送一也照樣喝星巴克？",
    "你會不會沒有折價券也直接叫 Uber？",
    // ... 更多問題
];
```

### 步驟 2: 新增問題

在陣列中加入新的問題字串。問題格式建議：

- 使用「你會不會...？」或「你...嗎？」的句型
- 問題應該是二選一的 Yes/No 類型
- 回答「對，就是我」會增加分數
- 回答「不，我不會」不會增加分數

範例：
```javascript
const questions = [
    "你會不會沒有買一送一也照樣喝星巴克？",
    "你會不會沒有折價券也直接叫 Uber？",
    "你會買名牌包嗎？", // 新增的問題
    "你出國旅遊都住五星級飯店嗎？", // 新增的問題
    // ... 其他問題
];
```

### 步驟 3: 更新進度顯示（可選）

如果您修改了問題總數，進度條和結果計算會自動更新，無需額外調整。

## 評分標準

測驗會根據回答「對，就是我」的題目數量計算百分比：

- 80% 以上：頂級台女認證 🔥
- 60-79%：準台女實習生 💎
- 40-59%：小資台女萌芽中 ✨
- 20-39%：省錢達人 🌱
- 20% 以下：終極省錢王 💰
